# DevOps End-to-End Project

## Project Overview

This project demonstrates a complete DevOps workflow for deploying a Python Flask application using:

- Git & GitHub
- Jenkins
- Docker
- Terraform
- AWS
- Ansible
- Kubernetes
- Prometheus
- Grafana

## Tech Stack

- Python Flask
- Git
- Jenkins
- Docker
- Kubernetes
- Terraform
- AWS EC2
- Ansible
- Prometheus
- Grafana
